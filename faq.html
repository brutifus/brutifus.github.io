FAQ¶

Q: What’s with brutus? Is it the same as brutifus?

A: the brutus code is the ancestor to brutifus, and is now deprecated. It may be that not all the functionalities of brutus will be implemented inside brutifus: it’ll all depend on what I use brutifus for.

Q: I’m getting some weird LaTeX error message … what is going on ?

A: If you set systemtex: True in params_brutifus.yaml, the code assumes that it is being run on a machine with a proper [citation needed] system-wide LaTeX installation, which should lead to prettier plots. If this fails for you, you can a) fix your LaTeX installation, or b) disable its use in favor of matplotlib’s native one by setting systemtex: False.
